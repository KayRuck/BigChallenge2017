Was kann unsere Lösung?

Unser Feature soll dem Nutzer ermöglichen technische Daten abzufragen. Dies soll so benutzerfreundlich wie möglich sein, weshalb wir in unsern Konzept eine einfache Bedienung mithilfe von Buttons und Icons verwenden. Damit die technischen Daten leicht verständlich und übersichtlich sind, werden hier Grafiken und Diagramme zur Veranschaulichung verwendet.

Energieanzeige

Hier findet der Nutzer alle Informationen zum Energiestand, Stromverbrauch und weiteren.

Fahrzeugdaten

Alle technischen Daten. (Reifendruck, Umdrehungszahl des Motors, PS-Anzeige, Bremsen)

Spurverhalten

Das Spurverhalten des Autos wird wiedergespiegelt (Reifenauslastung, Drehmoment).

Allgemeine Daten

Alle nützlichen Informationen (wann muss das Auto das nächste Mal zum Tüff, Breite, Höhe, etc ...)

Projektfortschritt

Unser Feature
Das Feature zeigt mithilfe eine Navigation, die mit Icons bedienbar ist, verschieden Daten an. Es wird unter vier Überpunkte unterschieden: Energieanzeige, Fahrzeugdaten, Spurverhalten und allgemeine Daten.
Die Daten sollen mithilfe von D3-Grafiken auf den Tablet angezeigt werden. Zur Bedienung wird die Navigation benutzt, jeder der Überpunkte wird ein Icon zugeordnet. Mithilfe eines Touchclicks werden dann die einzelnen Grafiken dargestellt.
